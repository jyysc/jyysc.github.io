>这是图像处理中灰度变换的一些基本操作（大牛直接忽略），也是数字图像处理课程中老师布置的作业，使用的是matlab编写，图片是matlab自带的coins.png，请多指教！ --- ##二值化

       图像二值化简单说就是把一张图像装换成二值图像。这里是使用matlab中的`graythresh`得到一个阈值，用阈值将图像的数据分成两部分：大于阈值的像素群和小于阈值的像素群。 ```matlab clear all; I = imread('.\coins.png'); D = []; I = im2double(I); thresh = graythresh(I); for i = 1:size(I,1) for j = 1:size(I,2) if I(i,j) <= thresh D(i,j) = 0; else D(i,j) = 1; end end end imshow(D); ``` ##反转

      使用反转变换，可得到灰度级范围为$[0,L-1]$的一幅图像的反转图像，该反转图像由下式给出：$$s=L-1-r$$使用这种方式反转一幅图像的灰度级，可得到等效的照片底片。这种类型的处理特别适用于增强嵌入在一幅图像的暗区域中的白色或灰色细节，特别是当黑色面积在尺寸上占主导地位时。 ```matlab clear all; I = imread('.\coins.png'); D = []; I = im2double(I); for i = 1:size(I,1) for j = 1:size(I,2) D(i,j) = 1 - I(i,j); end end imshow(D); ``` ##对数变换

      对数变换的通用形式为$$s=c\log(1+r)$$其中$c$是一个常数，并假设$r\geq0$。该变换将输入中范围较窄的低灰度值映射为输出中较宽范围的灰度值，相反地，对高的输入灰度值也是如此。此例中$c=1$、$r=1$。 ```matlab clear all; I = imread('.\coins.png'); D = []; I = im2double(I); D = log(1 + I); imshow(D); ``` ##幂率（伽马）变换

      幂率变换的基本形式为$$s=cr^\gamma$$其中$c$和$\gamma$为正常数。当$\gamma>1$ 时，把输入的窄带暗值映射到宽带输出亮值；当 $\gamma<1$时，把输入高值映射为宽带 。此例中$\gamma=0.4$。 ```matlab clear all; I = imread('.\coins.png'); D = []; I = im2double(I); D = I .^ 0.4; imshow(D); ``` ##分段线性变换

       为了突出感兴趣的目标或灰度区间，相对抑制那些不感兴趣的目标或灰度区间，常采用分段线性变换法。 ```matlab clear all; I = imread('.\coins.png'); D = []; I = im2double(I); for i = 1:size(I,1) for j = 1:size(I,2) if I(i,j) * 255 <= 80 D(i,j) = 2.5 * I(i,j); elseif I(i,j) * 255 <= 180 D(i,j) = 1.5 * I(i,j); else D(i,j) = 0.5 * I(i,j); end end end imshow(D); ``` --- >matlab将图像中的数据存储为64位浮点数（double）和无符号整数（uint8和uint16）。uint型的优势在于节省空间，涉及运算时要用im2double把图像数据转换成double型。uint8的操作仅仅是将一个double类型的小数点后面的部分全部去掉，但是im2uint8是将输入中所有小于0的数设置为0，而将输入中所有大小1的数值设置为255，再将其他数值乘以255。如果转换前的数据符合图像数据标准（0~1），那么可以直接使用im2uint8，否则使用uint8。

二值化

       图像二值化简单说就是把一张图像装换成二值图像。这里是使用matlab中的graythresh得到一个阈值，用阈值将图像的数据分成两部分：大于阈值的像素群和小于阈值的像素群。

clear all;
I = imread('.\coins.png');
D = [];
I = im2double(I);
thresh = graythresh(I); 
for i = 1:size(I,1)
    for j = 1:size(I,2)
        if I(i,j) <= thresh
            D(i,j) = 0;
        else
            D(i,j) = 1;
        end
    end
end
imshow(D);

反转

      使用反转变换，可得到灰度级范围为[0,L−1]的一幅图像的反转图像，该反转图像由下式给出：

s=L−1−r
使用这种方式反转一幅图像的灰度级，可得到等效的照片底片。这种类型的处理特别适用于增强嵌入在一幅图像的暗区域中的白色或灰色细节，特别是当黑色面积在尺寸上占主导地位时。

clear all;
I = imread('.\coins.png');
D = [];
I = im2double(I);
for i = 1:size(I,1)
    for j = 1:size(I,2)
       D(i,j) = 1 - I(i,j);
    end
end
imshow(D);

对数变换

      对数变换的通用形式为

s=clog(1+r)
其中c是一个常数，并假设r≥0。该变换将输入中范围较窄的低灰度值映射为输出中较宽范围的灰度值，相反地，对高的输入灰度值也是如此。此例中c=1、r=1。

clear all;
I = imread('.\coins.png');
D = [];
I = im2double(I);
D = log(1 + I);
imshow(D);

幂率（伽马）变换

      幂率变换的基本形式为

s=crγ
其中c和γ为正常数。当γ>1 时，把输入的窄带暗值映射到宽带输出亮值；当 γ<1时，把输入高值映射为宽带 。此例中γ=0.4。

clear all;
I = imread('.\coins.png');
D = [];
I = im2double(I);
D = I .^ 0.4;
imshow(D);

分段线性变换

       为了突出感兴趣的目标或灰度区间，相对抑制那些不感兴趣的目标或灰度区间，常采用分段线性变换法。

clear all;
I = imread('.\coins.png');
D = [];
I = im2double(I);
for i = 1:size(I,1)
    for j = 1:size(I,2)
        if I(i,j) * 255 <= 80
            D(i,j) = 2.5 * I(i,j);
        elseif I(i,j) * 255 <= 180
            D(i,j) = 1.5 * I(i,j);
        else
            D(i,j) = 0.5 * I(i,j);
        end
    end
end
imshow(D);
